# Missing Icons

The following PWA icons are referenced in `site.webmanifest` but are missing:

- `android-chrome-192x192.png` (192x192px)
- `android-chrome-512x512.png` (512x512px)

## To Fix:

1. Create or obtain PNG versions of your logo in these sizes
2. Place them in the `public/` directory
3. Rebuild the app

## Quick Fix:

You can use an online tool to generate these from your logo:
- https://realfavicongenerator.net/
- Or use https://favicon.io/

The theme color in the manifest is set to `#f59e0b` (amber-500).
